## 📄 Lexer

El lexer ya es capaz de identificar:

### 🔑 Palabras clave

- `var`
- `const`
- `si`
- `sino`
- `segun`
- `caso`
- `otro`
- `clase`
- `continuar`
- `fun`
- `falso`
- `mientras`
- `no`
- `nulo`
- `o`
- `para`
- `ret`
- `romper`
- `verdadero`
- `y`

### 🔢 Literales numéricos

- Números enteros: `2`, `897`, `1993`
- Números flotantes: `3.1415`, `4436.8`, `0.0`

### 🔠 Cadenas de texto

- Con comillas simples: `'hola'`
- Con comillas dobles: `"mundo"`

### 🔣 Delimitadores

- Paréntesis: `(`, `)`
- Corchetes: `[`, `]`
- Llaves: `{`, `}`
- Coma: `,`
- Punto: `.`
- Dos puntos: `:`
- Punto y coma: `;`

### 🧮 Operadores aritméticos 

- Suma: `+`
- Resta: `-`
- Multiplicación: `*`
- División: `/`
- Módulo: `%`

### ➡️ Operadores de asignación

- Igual: `=`

### 🟰 Operadores comparativos

- Menor que: `<`
- Mayor que: `>`
- Igualdad: `==`
- Distinto: `!=`
- Menor o igual: `<=`
- Mayor o igual: `>=`
